Open this link on your Apple Watch

This link is meant to be tapped on an Apple Watch that has SF Transit Watch installed. When it is, watchOS opens the app directly and saves your 511.org API key — you don't see this page at all.

If you're reading this on a phone or computer: forward the same link to yourself in Messages or Mail, open that message on your Apple Watch, and tap the link there.

The key is passed as the k query parameter. Nothing about this request is logged by this page.
